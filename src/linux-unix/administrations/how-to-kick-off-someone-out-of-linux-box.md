# [How to kick someone out of Linux box?](http://www.cyberciti.biz/tips/howto-linux-kill-and-logout-users.html)



Kick out a user:
```
pkill -9 -u username
```
Kick out a user from one session:
```
skill -KILL -v /dev/pts/*
```
